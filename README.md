# java-cli-sbt-h2-simple

## Description
A java sbt build, that connects to h2
database.

H2 is a database written in `java`
and is best known for a `dev` only
use.

## Tech stack
- java
- sbt
- h2 driver

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
